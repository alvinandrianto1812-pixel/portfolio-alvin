# Alvin Andrianto — Portfolio

Personal portfolio website built with Next.js 14, Three.js, 
and Framer Motion.

## Live Site
https://portfolio-alvin-ten.vercel.app

## Tech Stack
- Next.js 14 (App Router + TypeScript)
- Tailwind CSS
- Three.js + React Three Fiber (3D background)
- Framer Motion (animations)
- Deployed on Vercel

## Features
- 3D animated particle field with floating wireframe geometries
- Mouse parallax camera effect
- Custom cursor with trailing ring
- Scroll reveal animations
- 3D tilt effect on project cards
- Mobile responsive
- Security headers (CSP, X-Frame-Options, etc.)
